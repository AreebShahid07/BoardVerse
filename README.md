# BoardVerse 🎲

Welcome to **BoardVerse**, a premium suite of classic board games reimagined with a modern, retro-inspired aesthetic.

## 🚀 Features

- **Chess**: Fully functional chess engine with a sleek interface.
- **Reversi**: Classic strategy game with smooth animations.
- **Retro UI**: Powered by GSAP, Vanta.js, and Tone.js for an immersive experience.
- **Responsive Design**: Play on any device.

## 🛠️ Tech Stack

- **Framework**: React + Vite
- **Animations**: GSAP
- **3D/Visuals**: Vanta.js (WAVES)
- **Audio**: Tone.js
- **Logic**: chess.js, custom game engines

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone git@github.com:AreebShahid07/BoardVerse.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run locally:
   ```bash
   npm run dev
   ```
